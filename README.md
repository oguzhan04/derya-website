# Derya Website

Static website for Derya AI and Mailport.

## Getting Started

### First Time Setup

1. Clone the repository:
```bash
git clone https://github.com/oguzhan04/derya-website.git
cd derya-website
```

### Running Locally

1. Pull the latest changes (if working with others):
```bash
git pull
```

2. Start a local server:
```bash
python3 -m http.server 8000
```

3. Open in your browser:
- Homepage: http://localhost:8000
- Mailport page: http://localhost:8000/mailport.html

### Making Changes

1. Make your changes to the files
2. Stage your changes:
```bash
git add .
```

3. Commit:
```bash
git commit -m "Your commit message"
```

4. Push:
```bash
git push
```

## Pages

- `index.html` - Main homepage
- `mailport.html` - Mailport landing page
